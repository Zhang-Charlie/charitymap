# CharityMap web application

The public application uses Next.js, TypeScript, Tailwind CSS and MapLibre GL JS.

```bash
npm install
npm run dev
```

The app builds without a map provider key. Add `NEXT_PUBLIC_MAP_STYLE_URL` to `.env.local` when you are ready to use a hosted MapLibre-compatible style.

Validation:

```bash
npm run check
```
