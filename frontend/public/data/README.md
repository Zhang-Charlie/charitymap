# Country boundaries

`countries-110m.geojson` is derived from the Natural Earth 1:110m Admin 0 countries dataset.

- Source: https://github.com/nvkelso/natural-earth-vector
- Source file: `geojson/ne_110m_admin_0_countries.geojson`
- Retrieved: 2026-07-19
- License: public domain

The frontend copy retains each feature's geometry, ISO alpha-2 country code, English country name, and Natural Earth's cartographic label point. Label points position country-level aggregate badges and are not treated as event coordinates.
